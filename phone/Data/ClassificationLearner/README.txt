Gesture Classification Dataset Documentation
=======================================

Dataset Information:
-------------------
Total features: 56
Label encoding: 0-based indices (0, 1, 2, 3)

Using the Classification Learner App:
--------------------------------
1. In MATLAB, click on the "Apps" tab
2. Click on "Classification Learner"
3. Click "New Session"
4. Select the file "gesture_features.mat"
5. In the import dialog, select "Label" as the response variable and the others as predictors
6. Choose cross-validation options (5-fold recommended)
7. Click "Start Session"
8. Try different classification methods

Exporting the Model:
------------------
1. Once you find the best performing model, click "Export"
2. Choose "Export Model"
3. Save the model to your workspace or as a MATLAB file
